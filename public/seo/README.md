# SEO asset placeholders

Add future public SEO and social-sharing assets here.

Suggested files:

- `og-default.png` — default Open Graph / Twitter preview image
- `apple-touch-icon.png` — iOS home screen icon
- `icon-192.png` — optional web app icon
- `icon-512.png` — optional web app icon

Recommended notes:

- Keep filenames stable so metadata does not need constant updates.
- Prefer absolute production-safe references through Astro `site` metadata.
- Do not reference files from layout metadata until the real asset exists.
